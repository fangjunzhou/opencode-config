# Utility Scripts

This directory contains utility scripts used by agents and build processes.

## Available Scripts

- Setup scripts for environment configuration
- Build helpers
- Testing utilities
- etc.

## For Developers

Scripts in this directory should be:

- Portable (work on macOS, Linux, Windows)
- Well-documented with comments
- Include error handling
- Have clear usage instructions

Language variants can add their own utilities in `../python/scripts/`, etc.
